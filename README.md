# 📊 Predictive Sales Forecast

## Internship Information

- **Internship ID:** CITS5671

## Project Overview

This project is a **Predictive Sales Forecast Dashboard** developed using **Microsoft Power BI**. It analyzes historical sales data to identify trends, monitor business performance, and forecast future sales. The interactive dashboard helps users gain valuable insights for making informed business decisions through visual analytics and forecasting techniques.

## Features

- 📈 Interactive Dashboard
- 📊 KPI Cards (Sales, Profit, Orders, etc.)
- 📅 Sales Trend Analysis
- 🔮 Forecast Visualization
- 📦 Product & Category Performance Analysis
- 🎛️ Interactive Filters and Slicers
- 📉 Historical Sales Analysis
- 💡 Business Insights Dashboard

## Tools Used

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Data Visualization
- Sales Forecasting

## Project Objectives

- Analyze historical sales performance.
- Predict future sales trends using forecasting.
- Create an interactive business dashboard.
- Support data-driven decision-making.
- Improve business performance analysis through visual reports.
